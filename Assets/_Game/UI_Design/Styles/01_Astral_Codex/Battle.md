# Battle (Combat HUD) - Astral Codex Style

## Visual Aesthetic
- **Environment HUD**: Translucent glass frames for unit portraits. Corners are anchored with glowing golden constellations.
- **Grid Tiles**: A lattice of thin gold light on a deep indigo void.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnPause` | II Icon | Pause Menu | Rotating pulsar icon. |
| `BtnSpeed` | >> Icon | N/A (Toggle) | Streaking comets for 2x/3x speed. |
| `BtnAuto` | AUTO | N/A (Toggle) | A glowing lunar eye. |
| `BtnUnitIcon` | Unit Avatar | Deploy Unit | Glass-encased portrait with star particles. |
| `BtnSkill` | Skill Icon | Activate Skill | A miniature supernova that flares when ready. |

## Internal Windows & Overlays
- **Pause Menu**: A centered glass slab with "Stardust" blur effects.
- **Victory Screen**: Character portrait surrounded by a radiant golden aurora.
- **Defeat Screen**: Fading starfield with soft indigo shadows.

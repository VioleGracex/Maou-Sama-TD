# Conquest (Campaign Map) - Astral Codex Style

## Visual Aesthetic
- **Map Background**: A shifting, high-resolution nebula where the terrain is formed by gas clouds and star clusters.
- **Nodes**: Brilliantly glowing stars or miniature planets.
- **Connectors**: Faint, glowing ley lines that look like constellations.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A shooting star pointing left. |
| `BtnDiffToggle` | Difficulty | N/A (Internal) | A prism that changes color (Blue to Gold to Purple). |
| `BtnStageNode` | Stage Number | Tactical Prep | A pulsing star surrounded by a golden ring. |
| `BtnRegionRewards`| Chest Icon | Rewards Popup | A cosmic egg or crystal container. |

## Internal Windows & Overlays
- **Stage Detail Overlay**: A floating holographic glass panel that emerges from the star node.
- **Difficulty Popup**: Three orbs (Mortal, Heroic, Divine) that orbit each other until selected.
- **Tactical Prep Window**: A top-down view of a starlit battlefield on a translucent glass table.

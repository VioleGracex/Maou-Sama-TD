# Inspector - Astral Codex

## Visual Aesthetic
- **Atmosphere**: Charstand stands within a holographic circle of constellations.
- **Stat Display**: Thin-line gold icons floating over deep indigo glass.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Celestial Stats** | Thin-line moon/star icons for ATK, DEF, HP, etc. |
| **Runic Skills** | Circular glass frames for skill icons. |
| **Constellation Class** | Star-pattern icons representing the unit's tactical role. |

# Level Up - Astral Codex

## Visual Aesthetic
- **Atmosphere**: A shifting nebula background.
- **Mechanic**: Consuming "Stardust" shards to advance unit constellations.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Star Shard** | Duplicate unit token (Shard) for upgrade. |
| **Astral Meter** | XP bar with shimmering stardust fill. |
| **Ascend Button** | Gold-rimmed glass button for Level Up action. |

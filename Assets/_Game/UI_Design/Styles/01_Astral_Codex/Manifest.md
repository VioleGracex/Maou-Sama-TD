# Manifest (Summon Altar) - Astral Codex Style

## Visual Aesthetic
- **Background**: A deep-space ritual circle floating within a swirling nebula.
- **Summon Altar**: A crystalline prism that refracts starlight from the surrounding nebula.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Crystalline comet icon. |
| `BtnPull1` | Summon x1 | Summon Sequence| Single floating star shard icon. |
| `BtnPull10` | Summon x10 | Summon Sequence| A glowing galaxy spiral icon. |
| `BtnDetails` | Information | Details Window | Soft, glowing nebula-script icon. |
| `BtnBannerNext` | Right Arrow | Next Banner | Pulsing blue light ray. |

## Internal Windows & Overlays
- **Summon Result Grid**: 2x5 Grid of translucent glass cards. Stars flare when rarity is higher.
- **Character Reveal**: Screen explodes in a "Supernova" of blue and gold, revealing the character.
- **Pity Tracker**: A glowing constellation that completes itself as pulls are made.

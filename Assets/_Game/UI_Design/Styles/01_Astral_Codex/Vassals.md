# Vassals (Gallery) - Astral Codex Style

## Visual Aesthetic
- **Background**: An ethereal celestial archive where units are projected as star-charts.
- **Grid Layout**: Units are shown as glowing crystals or cards with drifting star particles.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Shooting star icon. |
| `BtnUnitCard` | Unit Portrait | Unit Details | Bordered by glowing runic glass panels. |
| `BtnFilter` | Funnel Icon | Filter Panel | A crystal prism icon. |
| `BtnBatchAction`| Star Icon | Batch Selection| A constellation cluster icon. |

## Internal Windows & Overlays
- **Filter Panel**: Slides from the top; a starry nebula field with constellations for filters.
- **Unit Details Overlay**: Character stands on a rotating starfield. Stats are displayed in thin gold networks.
- **Upgrade Window**: A sub-page where units are placed in celestial orbits to be empowered.

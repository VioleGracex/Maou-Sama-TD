# Battle (Combat HUD) - Abyssal Forge Style

## Visual Aesthetic
- **Environment HUD**: Heavy riveted iron plates for unit portraits at the bottom.
- **Grid Tiles**: Scratched charred metal with glowing orange lava seams.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnPause` | II Icon | Pause Menu | Heavy metallic cog icon. |
| `BtnSpeed` | >> Icon | N/A (Toggle) | Rotating iron gears for speed tiers. |
| `BtnAuto` | AUTO | N/A (Toggle) | A flaming furnace eye icon. |
| `BtnUnitIcon` | Unit Avatar | Deploy Unit | Iron-framed portrait with soot textures. |
| `BtnSkill` | Skill Icon | Activate Skill | A bursting magma furnace that glows white-hot when ready. |

## Internal Windows & Overlays
- **Pause Menu**: Heavy metal shutter that slams down to show settings.
- **Victory Screen**: Large "VICTORY" text hammered out of red-hot metal.
- **Defeat Screen**: Shattered iron plate effect with cool grey smoke.

# Conquest (Campaign Map) - Abyssal Forge Style

## Visual Aesthetic
- **Map Background**: A charred, topographical view of a volcanic region, etched into a blackened iron table.
- **Nodes**: Burning craters, iron towers, or demonic fortresses.
- **Connectors**: Deep grooves filled with flowing molten lava.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A jagged iron spike pointing left. |
| `BtnDiffToggle` | Difficulty | N/A (Internal) | A furnace bellows icon that changes the glow intensity. |
| `BtnStageNode` | Stage Number | Tactical Prep | A glowing orange crater on the iron map. |
| `BtnRegionRewards`| Chest Icon | Rewards Popup | An iron cage containing glowing ingots. |

## Internal Windows & Overlays
- **Stage Detail Overlay**: A heavy metal door that slams down, revealing mission data in orange text.
- **Difficulty Popup**: Three furnace heat levels (Smolder, Blaze, Infernal) displayed as gauges.
- **Tactical Prep Window**: A rugged iron interface where units are "clicked" into heavy slots.

# Inspector - Abyssal Forge

## Visual Aesthetic
- **Atmosphere**: Character stands on a glowing iron platform amid steam and sparks.
- **Stat Display**: Rugged iron frames with mechanical details.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Gauge Stats** | Dial or pressure-gauge icons for ATK, DEF, HP, etc. |
| **Smelted Skills** | Heavy iron square frames for skill icons. |
| **Crest Class** | Hammered iron emblems for unit categories. |

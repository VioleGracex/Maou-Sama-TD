# Level Up - Abyssal Forge

## Visual Aesthetic
- **Atmosphere**: A volcanic anvil where characters are "re-forged".
- **Mechanic**: Consuming "Soul Ingots" to increase heat levels.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Soul Ingot** | Iron-bound token for duplicate characters. |
| **Lava Meter** | XP bar resembling a glass tube filled with molten lava. |
| **Forge Button** | Riveted metal plate button to Level Up. |

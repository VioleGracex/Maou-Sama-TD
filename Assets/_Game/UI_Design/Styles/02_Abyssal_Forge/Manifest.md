# Manifest (Summon Altar) - Abyssal Forge Style

## Visual Aesthetic
- **Background**: A deep, heat-hazed smithy with giant bellows and iron chains hanging.
- **Summon Altar**: A colossal iron anvil surrounded by white-hot lava channels.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Jagged iron shard. |
| `BtnPull1` | Summon x1 | Summon Sequence| Single iron-ingot icon. |
| `BtnPull10` | Summon x10 | Summon Sequence| A heavy cluster of glowing iron bars. |
| `BtnDetails` | Information | Details Window | Engraved iron tablet icon. |
| `BtnBannerNext` | Right Arrow | Next Banner | A blast of orange sparks. |

## Internal Windows & Overlays
- **Summon Result Grid**: 2x5 Grid of heavy obsidian slabs. 
- **Character Reveal**: Character is hammered out of white-hot metal, cooling in front of the camera.
- **Pity Tracker**: A pressure gauge that rises with each pull; steam bursts at 100%.

# Vassals (Gallery) - Abyssal Forge Style

## Visual Aesthetic
- **Background**: An industrial barracks with shelves of iron-framed display plates.
- **Grid Layout**: Heavy obsidian cards with glowing orange numbers and riveted borders.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Iron spike icon. |
| `BtnUnitCard` | Unit Portrait | Unit Details | Heavy iron frame with glowing cracks. |
| `BtnFilter` | Gear Icon | Filter Panel | A rotating, jagged iron gear icon. |
| `BtnBatchAction`| Hammer Icon | Batch Selection| A heavy smithing hammer icon. |

## Internal Windows & Overlays
- **Filter Panel**: Slides from the bottom like a metal shutter.
- **Unit Details Overlay**: Character stands on an iron platform. Stats shown as furnace gauges.
- **Upgrade Window**: A sub-page featuring an anvil UI where units are leveled using iron shards.

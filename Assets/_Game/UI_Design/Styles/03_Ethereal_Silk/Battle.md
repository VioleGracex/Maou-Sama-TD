# Battle (Combat HUD) - Ethereal Silk Style

## Visual Aesthetic
- **Environment HUD**: Clean, minimalist semi-transparent ribbons for unit portraits.
- **Grid Tiles**: Faint ink-wash lines on a textured paper surface.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnPause` | II Icon | Pause Menu | Simple brush-stroke "Zen" circle. |
| `BtnSpeed` | >> Icon | N/A (Toggle) | Floating petals for speed indication. |
| `BtnAuto` | AUTO | N/A (Toggle) | An ink-eye calligraphy symbol. |
| `BtnUnitIcon` | Unit Avatar | Deploy Unit | Circular soft-focus portrait. |
| `BtnSkill` | Skill Icon | Activate Skill | A blooming flower or expanding ink-wash that fills when ready. |

## Internal Windows & Overlays
- **Pause Menu**: A soft vellum layer that blurs the battlefield.
- **Victory Screen**: Elegant calligraphy "TRIUMPH" over a serene mountain view.
- **Defeat Screen**: Faded ink-splatter effect with falling leaves.

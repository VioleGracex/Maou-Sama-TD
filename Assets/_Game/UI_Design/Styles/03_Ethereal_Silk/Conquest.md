# Conquest (Campaign Map) - Ethereal Silk Style

## Visual Aesthetic
- **Map Background**: An expansive ink-wash painting of mountains and rivers on a giant horizontal scroll.
- **Nodes**: Stamp-style calligraphy marks or blooming lotus flowers.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A soft brush stroke pointing left. |
| `BtnDiffToggle` | Difficulty | N/A (Internal) | A layered silk ribbon that changes color shades. |
| `BtnStageNode` | Stage Number | Tactical Prep | A glowing paper lantern on the map. |
| `BtnRegionRewards`| Box Icon | Rewards Popup | A minimalist jade box with a soft glow. |

## Internal Windows & Overlays
- **Stage Detail Overlay**: A scrolls unfurls from the center of the node, showing mission text.
- **Difficulty Popup**: Three fans (Zen, Flow, Storm) that open in sequence.
- **Tactical Prep Window**: A serene paper-textured layout with soft-focus unit slots.

# Inspector - Ethereal Silk

## Visual Aesthetic
- **Atmosphere**: Unit framed by falling cherry blossoms and silk ribbons.
- **Stat Display**: Minimalist text on semi-transparent vellum.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Zen Stats** | Thin-line hand-drawn symbols for ATK, DEF, HP, etc. |
| **Vellum Skills** | Soft rounded silk-textured frames for skills. |
| **Stamp Class** | Traditional red ink-stamp icons for roles. |

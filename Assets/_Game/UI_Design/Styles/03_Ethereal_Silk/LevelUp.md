# Level Up - Ethereal Silk

## Visual Aesthetic
- **Atmosphere**: An ink-wash scroll that fills with color as levels increase.
- **Mechanic**: Painting characters into higher tiers with "Ink Essence".

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Ink Droplet** | Delicate brush-stroke token for duplicates. |
| **Silk Meter** | Flowing ribbon XP bar with jade color wash. |
| **Brush Button** | Calligraphic stroke button to Commit Level Up. |

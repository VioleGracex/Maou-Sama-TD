# Manifest (Summon Altar) - Ethereal Silk Style

## Visual Aesthetic
- **Background**: A tranquil garden with a floating brush that paints on air.
- **Summon Altar**: A simple jade bowl filled with glowing swirling ink.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A falling petal icon. |
| `BtnPull1` | Summon x1 | Summon Sequence| Single floating silk ribbon. |
| `BtnPull10` | Summon x10 | Summon Sequence| Ten blooming cherry blossoms icon. |
| `BtnDetails` | Information | Details Window | Soft ink-wash scroll icon. |
| `BtnBannerNext` | Right Arrow | Next Banner | A ripple effect in the background. |

## Internal Windows & Overlays
- **Summon Result Grid**: 2x5 Grid of silk textured cards. Rarity shown by color washes.
- **Character Reveal**: Character is painted into view by a flurry of ink and petals.
- **Pity Tracker**: A tea cup that slowly fills with jade liquid.

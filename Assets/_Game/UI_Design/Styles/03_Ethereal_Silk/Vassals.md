# Vassals (Gallery) - Ethereal Silk Style

## Visual Aesthetic
- **Background**: A tranquil, minimalist archive with soft-focus cherry blossoms and silk banners.
- **Grid Layout**: Units are displayed in delicate silk-textured cards with soft ink-wash backgrounds.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Soft brush stroke icon. |
| `BtnUnitCard` | Unit Portrait | Unit Details | Bordered by flowing silk ribbon effects. |
| `BtnFilter` | Fan Icon | Filter Panel | A hand-painted fan icon. |
| `BtnBatchAction`| Lotus Icon | Batch Selection| A blooming lotus flower icon. |

## Internal Windows & Overlays
- **Filter Panel**: A transparent vellum layer that floats into view.
- **Unit Details Overlay**: Character stands in a misty garden spotlight. Stats are vertical calligraphy.
- **Upgrade Window**: A sub-page where units move through a "Zen Garden" ritual UI.

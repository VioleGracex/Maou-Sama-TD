# Battle (Combat HUD) - Blood Contract Style

## Visual Aesthetic
- **Environment HUD**: Elaborate silver frames with red velvet accents around the corners.
- **Grid Tiles**: Etched stone with blood-red energy filling the cracks when a unit is selected for deployment.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnPause` | II Icon | Pause Menu | Silver stopwatch with a ruby face. |
| `BtnSpeed` | >> Icon | N/A (Toggle) | Silver daggers that multiply based on speed (1, 2, 3). |
| `BtnAuto` | AUTO | N/A (Toggle) | Glowing silver eye icon. |
| `BtnUnitIcon` | Unit Avatar | Deploy Unit | Portrait in a circular silver frame. |
| `BtnSkill` | Skill Icon | Activate Skill | Large silver chalice or ritual seal that "blooms" when ready. |

## Internal Windows & Overlays
- **Pause Menu**: A centered dark marble slab with options for "Retreat" (broken dagger) and "Retry" (looping vine).
- **Victory Screen**: Large "BLOOD CONTRACT FULFILLED" text in silver, with crimson light rays.
- **Defeat Screen**: Cracked red mirror effect with fading silver borders.

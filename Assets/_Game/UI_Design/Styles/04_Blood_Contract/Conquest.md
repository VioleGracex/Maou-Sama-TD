# Conquest (Campaign Map) - Blood Contract Style

## Visual Aesthetic
- **Map Background**: An ancient, blood-stained map of a dark empire spread across a cracked marble table.
- **Nodes**: Gothic cathedrals, crumbling forts, and tomb-like structures.
- **Connectors**: Dark red "veins" that pulse with light between unlocked nodes.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Silver dagger pointing left. |
| `BtnDiffToggle` | Difficulty | N/A (Internal) | A silver slider that turns red when "Hard" is selected. |
| `BtnStageNode` | Stage Number | Tactical Prep | A glowing silver seal on the map. |
| `BtnRegionRewards`| Chest Icon | Rewards Popup | A silver casket with red velvet lining. |

## Internal Windows & Overlays
- **Stage Detail Overlay**: A full-screen "Tombstone" slab that rises from the bottom. Shows enemies and rewards in silver-etched text.
- **Difficulty Popup**: A set of three silver cards (Normal, Hard, Hell) that fan out.
- **Tactical Prep Window**: A dark marble layout where units are placed in silver-rimmed slots.

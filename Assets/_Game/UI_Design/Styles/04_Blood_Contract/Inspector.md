# Inspector - Blood Contract

## Visual Aesthetic
- **Atmosphere**: Unit stand in a spotlight of red light within a dark gallery.
- **Stat Display**: Silver plaques with sharp crimson etchings.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Sanguine Stats** | Sharp geometric icons (Sword, Shield) in crimson. |
| **Vial Skills** | Circular vials or silver-rimmed frames for skills. |
| **Mask Class** | Ritualized mask icons identifying tactical roles. |

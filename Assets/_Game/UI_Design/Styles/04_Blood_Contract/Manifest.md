# Manifest (Summon Altar) - Blood Contract Style

## Visual Aesthetic
- **Background**: A ritual chamber with silver candles and a large central altar.
- **Summon Altar**: A silver chalice that fills with glowing crimson liquid during the ritual.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Silver filigree dagger. |
| `BtnPull1` | Summon x1 | Summon Sequence| Silver ritual dagger inscribed with "One Soul". |
| `BtnPull10` | Summon x10 | Summon Sequence| Silver chalice icon inscribed with "Ten Souls". |
| `BtnDetails` | Information | Details Window | A small silver scroll with a red ribbon. |
| `BtnBannerNext` | Right Arrow | Next Banner | Red "Bleed" arrow pointing right. |

## Internal Windows & Overlays
- **Summon Result Grid**: A 2x5 grid of stone-textured cards. Rare units have silver or gold filigree highlights.
- **Character Reveal**: Full-screen art of the character emerging from a spray of crimson petals and silver light.
- **Pity Tracker**: A silver vial that fills with red liquid as pulls are made.

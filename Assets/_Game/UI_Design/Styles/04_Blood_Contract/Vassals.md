# Vassals (Gallery) - Blood Contract Style

## Visual Aesthetic
- **Background**: A grand portrait gallery in a dark Gothic castle, lit by flickering silver candles.
- **Grid Layout**: Unit cards are displayed in heavy silver frames on dark velvet backgrounds.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Silver filigree dagger. |
| `BtnUnitCard` | Unit Portrait | Unit Details | Portrait framed in silver ivy or thorns. |
| `BtnFilter` | Funnel Icon | Filter Panel | A silver ritual mask icon. |
| `BtnBatchDisposal`| Trash Icon | Batch Selection| A silver sacrificial bowl icon. |

## Internal Windows & Overlays
- **Filter Panel**: Slides from the left; a dark marble slab with silver tick-boxes.
- **Unit Details Overlay**: A full-screen view where the character stands in a spotlight of red light. Stats are shown on silver plaques.
- **Upgrade Window**: A sub-panel featuring a silver ritual circle where materials are "sacrificed" for levels.

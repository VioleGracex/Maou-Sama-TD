# Battle (Combat HUD) - Runic Neon Style

## Visual Aesthetic
- **Environment HUD**: Glowing neon holographic borders for all elements. Flickering UI scan-lines.
- **Grid Tiles**: Blue laser lattice that glows brighter when a unit is placed.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnPause` | II Icon | Pause Menu | Digital "Loading" circle hexagon. |
| `BtnSpeed` | >> Icon | N/A (Toggle) | Blue neon arrows that increase in intensity at higher speeds. |
| `BtnAuto` | AUTO | N/A (Toggle) | Glowing CPU chip icon. |
| `BtnUnitIcon` | Unit Avatar | Deploy Unit | Digital portrait with scanning laser lines. |
| `BtnSkill` | Skill Icon | Activate Skill | A pulsing neon node that "overloads" when ready. |

## Internal Windows & Overlays
- **Pause Menu**: A terminal-style translucent box that pops into the center.
- **Victory Screen**: "CONNECTION SECURED" in bright neon green with binary fall.
- **Defeat Screen**: Static glitch effect with red "SYSTEM CRITICAL" error message.

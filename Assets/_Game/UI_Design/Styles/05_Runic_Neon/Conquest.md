# Conquest (Campaign Map) - Runic Neon Style

## Visual Aesthetic
- **Map Background**: A 3D topographical holographic scan of the world, with blue laser grid lines and flickering static edges.
- **Nodes**: Glowing neon cubes or data-shards.
- **Connectors**: High-speed fiber-optic style light lines.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A flickering cyan chevron pointing left. |
| `BtnDiffToggle` | Difficulty | N/A (Internal) | A digital switch that changes the map hue from blue to magenta. |
| `BtnStageNode` | Stage Number | Tactical Prep | A pulsing neon beacon. |
| `BtnRegionRewards`| Box Icon | Rewards Popup | A low-poly digital crate with a glowing core. |

## Internal Windows & Overlays
- **Stage Detail Overlay**: A window that "boots up" with a command-line interface effect.
- **Difficulty Popup**: "Safety", "Override", and "System Collapse" options in a terminal list.
- **Tactical Prep Window**: A top-down blueprint scan with unit avatars represented as digital tags.

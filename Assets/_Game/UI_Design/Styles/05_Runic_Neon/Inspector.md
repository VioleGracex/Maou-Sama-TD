# Inspector - Runic Neon

## Visual Aesthetic
- **Atmosphere**: Unit framed by an AR overlay and streaming data.
- **Stat Display**: Animated histograms and radar charts in neon cyan/magenta.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Vector Stats** | Circuitry-based symbols for ATK, DEF, HP, etc. |
| **Hex Skills** | Pulsing hexagonal frames for skill icons. |
| **Scan Class** | Digital HUD icons identifying unit roles. |

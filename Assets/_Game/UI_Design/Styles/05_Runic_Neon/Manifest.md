# Manifest (Summon Altar) - Runic Neon Style

## Visual Aesthetic
- **Background**: A high-tech "Manifestor" machine with laser arrays and spinning magnifiers.
- **Summon Altar**: A digital platform where characters are "reconstructed" from data streams.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Flickering cyan arrow. |
| `BtnPull1` | Summon x1 | Summon Sequence| Single digital core icon. |
| `BtnPull10` | Summon x10 | Summon Sequence| A data-packet cluster icon. |
| `BtnDetails` | Information | Details Window | Holographic "Help" icon. |
| `BtnBannerNext` | Right Arrow | Next Banner | Digital "Swipe" effect icon. |

## Internal Windows & Overlays
- **Summon Result Grid**: 2x5 Grid of flickering data-cards. High rarity causes "System Alert" red light effects.
- **Character Reveal**: A massive digital "Unzip" animation where the character loads from wireframe to full color.
- **Pity Tracker**: A digital progress bar that fills with magi-code.

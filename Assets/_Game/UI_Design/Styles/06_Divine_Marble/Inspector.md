# Inspector - Divine Marble

## Visual Aesthetic
- **Atmosphere**: Unit stands on a white marble platform lit by warm sunbeams.
- **Stat Display**: High-contrast serif text on beveled ivory side-panels.

| Icon Name | Usage |

| **Cornice Skills** | Gold-rimmed marble tiles for individual skills. |
| **Wing Class** | Golden wing motifs representing unit categories. |

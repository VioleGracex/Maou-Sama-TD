# Vassals (Gallery) - Divine Marble Style

## Visual Aesthetic
- **Background**: A serene museum of white marble and gold, where units are displayed as statues or framed reliefs.
- **Grid Layout**: Units are shown in beveled ivory frames with ornate golden cornices.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Golden feather icon. |
| `BtnUnitCard` | Unit Portrait | Unit Details | White marble frame with golden filigree. |
| `BtnFilter` | Scale Icon | Filter Panel | A golden balance scale icon. |
| `BtnBatchAction`| Scepter Icon | Batch Selection| A golden scepter icon. |

## Internal Windows & Overlays
- **Filter Panel**: A marble tablet that fades in from the right.
- **Unit Details Overlay**: Character stands on a sunlit pedestal. Stats are engraved in stone-look panels.
- **Upgrade Window**: A sub-page featuring a temple altar where units are "blessed" for levels.

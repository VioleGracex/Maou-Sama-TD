# Inspector - Monochrome Shade

## Visual Aesthetic
- **Atmosphere**: Unit displayed as a stark charcoal sketch on textured paper.
- **Stat Display**: Bold black/white blocks with minimal high-contrast text.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Silho Stats** | High-contrast black/white silhouette icons. |
| **Ink Skills** | Rough ink-blot frames around skill graphics. |
| **Scribble Class** | Abstract hand-drawn icons for unit roles. |

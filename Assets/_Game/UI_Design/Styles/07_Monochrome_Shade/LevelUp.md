# Level Up - Monochrome Shade

## Visual Aesthetic
- **Atmosphere**: A blank white canvas being filled with charcoal sketches.
- **Mechanic**: Using "Ink Splatters" to redefine the character's strength.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Doodle Item** | Rough charcoal-sketched token for duplicates. |
| **Hatch Meter** | XP bar using diagonal cross-hatch shading to fill. |
| **Sketch Button** | Hand-drawn thick-stroke rectangle for Level Up. |

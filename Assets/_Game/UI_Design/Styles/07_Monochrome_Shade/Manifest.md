# Manifest (Summon Altar) - Monochrome Shade Style

## Visual Aesthetic
- **Background**: A stark, white room where ink droplets fall from an invisible ceiling into a pool of black ink.
- **Summon Altar**: A simple, roughly-sketched stone well.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A sharp black ink-stroke arrow. |
| `BtnPull1` | Summon x1 | Summon Sequence| A single large ink droplet icon. |
| `BtnPull10` | Summon x10 | Summon Sequence| An ink-splatter cluster icon. |
| `BtnDetails` | Information | Details Window | A sketched "i" within a rough circle. |
| `BtnBannerNext` | Right Arrow | Next Banner | A quick horizontal pencil line. |

## Internal Windows & Overlays
- **Summon Result Grid**: 2x5 Grid of roughly-drawn rectangle cards on paper texture.
- **Character Reveal**: A massive ink splat that resolves into the character's high-contrast portrait.
- **Pity Tracker**: A sketched bottle filling with black ink.

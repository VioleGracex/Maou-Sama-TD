# Vassals (Gallery) - Monochrome Shade Style

## Visual Aesthetic
- **Background**: A chaotic sketchbook where units are taped or sketched onto different sections.
- **Grid Layout**: Units are shown as rough thumbnail sketches with pencil-drawn borders and hand-written names.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A hand-sketched arrow. |
| `BtnUnitCard` | Unit Portrait | Unit Details | Scrawled pencil border with charcoal shading. |
| `BtnFilter` | Eye Icon | Filter Panel | A rough sketched lens/eye icon. |
| `BtnBatchAction`| Lasso Icon | Batch Selection| A sketchy lasso icon. |

## Internal Windows & Overlays
- **Filter Panel**: A piece of "torn" paper that slides from the edge.
- **Unit Details Overlay**: Character is displayed in full-page charcoal detail. Stats are shown in hand-drawn bar graphs.
- **Upgrade Window**: A sub-page where units are "refined" with more detail or cross-hatching to show level.

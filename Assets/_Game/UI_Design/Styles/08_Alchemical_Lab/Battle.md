# Battle (Combat HUD) - Alchemical Laboratory Style

## Visual Aesthetic
- **Environment HUD**: Copper-rimmed glass frames with steam-venting effects.
- **Grid Tiles**: Polished brass lattice on a dark obsidian laboratory floor.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnPause` | II Icon | Pause Menu | A rotating brass barometer. |
| `BtnSpeed` | >> Icon | N/A (Toggle) | Spinning copper gears that accelerate with speed. |
| `BtnAuto` | AUTO | N/A (Toggle) | A glowing emerald test-tube eye. |
| `BtnUnitIcon` | Unit Avatar | Deploy Unit | Glass-encased portrait with liquid bubbles. |
| `BtnSkill` | Skill Icon | Activate Skill | A boiling beaker that "pops" its cork when ready. |

## Internal Windows & Overlays
- **Pause Menu**: A hinged copper plate that swings into view.
- **Victory Screen**: Large "EXPERIMENT SUCCESSFUL" in glowing green liquid text.
- **Defeat Screen**: Fading sparks and shattered glass effects.

# Conquest (Campaign Map) - Alchemical Laboratory Style

## Visual Aesthetic
- **Map Background**: An intricate world map drawn on yellowed parchment, pinned to a copper-edged laboratory desk.
- **Nodes**: Glass domes or copper-rimmed magnifying lenses over map regions.
- **Connectors**: Glowing green chemical trace lines.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A glass pipette icon pointing left. |
| `BtnDiffToggle` | Difficulty | N/A (Internal) | A three-position brass lever (Diluted, Concentrated, Pure). |
| `BtnStageNode` | Stage Number | Tactical Prep | A bubbling green test tube on the map. |
| `BtnRegionRewards`| Flask Icon | Rewards Popup | A glass flask containing glowing liquid. |

## Internal Windows & Overlays
- **Stage Detail Overlay**: A "Specimen Log" page that slides onto the screen.
- **Difficulty Popup**: Three colored vials (Yellow, Orange, Red) in a copper rack.
- **Tactical Prep Window**: A top-down blueprint view on a grid of hexagonal tiles.

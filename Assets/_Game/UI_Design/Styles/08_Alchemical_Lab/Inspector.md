# Inspector - Alchemical Lab

## Visual Aesthetic
- **Atmosphere**: Unit viewed through a large brass magnifying lens.
- **Stat Display**: Handwritten botanical-style entries on parchment tabs.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Apparatus Stats** | Gear, flask, and scale icons in copper/brass. |
| **Stopper Skills** | Glass-topped cork-rimmed frames for skill icons. |
| **Herb Class** | Distinct botanical sketches identifying unit types. |

# Level Up - Alchemical Lab

## Visual Aesthetic
- **Atmosphere**: A bubbling copper cauldron in a cluttered apothecary.
- **Mechanic**: Distilling "Emerald Fluid" into unit potency.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Flask Token** | Glass vial icon representing duplicate unit cards. |
| **Beaker Meter** | XP bar resembling a graded glass test tube. |
| **Mix Button** | Copper wheel or lever icon to initiate Level Up. |

# Manifest (Summon Altar) - Alchemical Laboratory Style

## Visual Aesthetic
- **Background**: A chaotic lab bench where a floating iron brush paints on air with glowing liquids.
- **Summon Altar**: A master cauldron surrounded by copper pipes that glow white-hot during the ritual.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Glass tube pointing left. |
| `BtnPull1` | Summon x1 | Summon Sequence| A glowing emerald vial. |
| `BtnPull10` | Summon x10 | Summon Sequence| A rack of ten bubbling test tubes. |
| `BtnDetails` | Information | Details Window | Stained parchment scroll icon. |
| `BtnBannerNext` | Right Arrow | Next Banner | A burst of green steam. |

## Internal Windows & Overlays
- **Summon Result Grid**: 2x5 Grid of glass-plated cards. Rarity is indicated by the color of liquid bubbling inside the frame.
- **Character Reveal**: A massive chemical reaction in the cauldron; character emerges as the green vapor clears.
- **Pity Tracker**: A barometer with a needle that moves toward an "Experiment Success" zone.

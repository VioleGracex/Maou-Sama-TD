# Vassals (Gallery) - Alchemical Laboratory Style

## Visual Aesthetic
- **Background**: An apothecary-style storage wall with hundreds of drawers and glass specimen cases.
- **Grid Layout**: Units are shown as botanical sketches pinned to dark mahogany panels with copper clips.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Copper needle icon. |
| `BtnUnitCard` | Unit Portrait | Unit Details | Copper-rimmed frame with a glass lens overlay. |
| `BtnFilter` | Flask Icon | Filter Panel | A bubbling Erlenmeyer flask icon. |
| `BtnBatchAction`| Tongs Icon | Batch Selection| A smithing tongs icon. |

## Internal Windows & Overlays
- **Filter Panel**: A sliding copper door that reveals a set of colored liquid switches.
- **Unit Details Overlay**: Character stands on a lab bench under a magnifying lens. Stats are handwritten on parchment labels.
- **Upgrade Window**: A sub-page featuring a chemical mixing UI where units are leveled using distilled essences.

# Battle (Combat HUD) - Void Weaver Style

## Visual Aesthetic
- **Environment HUD**: Shifting shadow-borders that pulse with purple light.
- **Grid Tiles**: Distorted fractal lattice on a black abyss floor.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnPause` | II Icon | Pause Menu | Closing pupil eye icon. |
| `BtnSpeed` | >> Icon | N/A (Toggle) | Streaking shadow-tendrils for speed. |
| `BtnAuto` | AUTO | N/A (Toggle) | A pulsing purple heart icon. |
| `BtnUnitIcon` | Unit Avatar | Deploy Unit | Shadow-shrouded portrait with glowing eyes. |
| `BtnSkill` | Skill Icon | Activate Skill | A reality-crack that glows magenta when ready. |

## Internal Windows & Overlays
- **Pause Menu**: A field of shadows that swallows the screen center.
- **Victory Screen**: "REALITY CONSUMED" in pulsing purple glyphs.
- **Defeat Screen**: A flash of white void followed by a fade to black.

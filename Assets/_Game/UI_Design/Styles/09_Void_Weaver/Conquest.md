# Conquest (Campaign Map) - Void Weaver Style

## Visual Aesthetic
- **Map Background**: A distorted view of the world as if seen through a warped glass orb. Continents are twisting.
- **Nodes**: Pulsing void rifts or blinking cosmic eyes.
- **Connectors**: Distorted, glowing shadow ribbons.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A shadow-smoke trail pointing left. |
| `BtnDiffToggle` | Difficulty | N/A (Internal) | A shadow-orb that darkens with higher difficulty. |
| `BtnStageNode` | Stage Number | Tactical Prep | A swirling black hole node. |
| `BtnRegionRewards`| Eye Icon | Rewards Popup | A blinking eye that opens to reveal rewards. |

## Internal Windows & Overlays
- **Stage Detail Overlay**: A shadow field that obscures the rest of the map, focusing on mission data.
- **Difficulty Popup**: "Stable", "Distorted", and "Forsaken" difficulty cards.
- **Tactical Prep Window**: A non-Euclidean grid where units are placed on pulsing void cells.

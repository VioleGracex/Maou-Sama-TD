# Inspector - Void Weaver

## Visual Aesthetic
- **Atmosphere**: Unit framed by creeping tendrils and distorted UI elements.
- **Stat Display**: Unconventional pulsing waveforms and alien glyphs.

## Icons & Usage
| Icon Name | Usage |
| :--- | :--- |
| **Fractal Stats** | Non-Euclidean geometric icons for ATK, DEF, etc. |
| **Maw Skills** | Shifting shadow-tendril frames for skill icons. |
| **Eldritch Class** | Alien eye-glyph icons identifying unit roles. |

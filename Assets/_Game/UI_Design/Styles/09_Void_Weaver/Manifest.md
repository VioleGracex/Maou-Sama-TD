# Manifest (Summon Altar) - Void Weaver Style

## Visual Aesthetic
- **Background**: A deep abyssal rift where stars are being consumed by shadow tendrils.
- **Summon Altar**: A black hole surrounded by pulsing purple ley lines.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | A shadow-smoke trail. |
| `BtnPull1` | Summon x1 | Summon Sequence| A single pulsing purple eye icon. |
| `BtnPull10` | Summon x10 | Summon Sequence| A cluster of shadow tentacles clutching a heart. |
| `BtnDetails` | Information | Details Window | A shifting fractal cube icon. |
| `BtnBannerNext` | Right Arrow | Next Banner | A tear in the void background. |

## Internal Windows & Overlays
- **Summon Result Grid**: 2x5 Grid of shifting fractal cards. Higher rarity cards have blinking eyes that watch the player.
- **Character Reveal**: Reality cracks into pieces; the character is pulled out of the resulting black hole.
- **Pity Tracker**: A shadowy heart that beats faster as pulls increase.

# Style Manifest: Void Weaver

## Core Visual Prompt
**Theme**: Eldritch, Cosmic Horror, Fractal Abyss.
**Colors**: Void Black (#000000), Pulsing Purple (#800080), Sickly Green (#32CD32).
**Aesthetic**: Shifting shadows, fractal patterns, and distorted UI elements.
**Typography**: "Jura" - angular, slightly alien, technical.

## Component Design
- **Buttons**: Amorphous blobs of shadow that crystallize when hovered.
- **Panels**: Fields of shifting fractals that "eat" the background.
- **Icons**: Eyes, tentacles, and non-Euclidean shapes that pulse.

## Page Flows (Gacha Optimized)

### Citadel (Home)
- **Character Frame**: Character partially obscured by creeping shadows and floating eyes.
- **UI Layout**: Floating, drifting menu elements. 
- **Interaction**: Whispering sound effects and low drones.

### Conquest (Campaign Map)
- **Map Style**: Distorted, warping view of the world from the "outside".
- **Navigation**: Reality-tearing full-screen overlays for mission details.
- **Nodes**: Pulsing rifts or blinking cosmic eyes.

### Manifest (Summon Altar)
- **Summon Sequence**: Screen cracks into fractal pieces to reveal the character.
- **Results Screen**: A distorted, pulsing Grid Scroll View (2x5).
- **Reveal**: Character is "pulled" from a black hole.

### Vassals (Gallery)
- **Layout**: Grid Scroll View where each card is a "captured" eldritch entity.
- **Filters**: Blinking eye icons for sorting by threat level (rarity).
- **Stats**: Unconventional "Life Sign" waveforms in side-panels.

### Battle (Tactical Grid)
- **Grid**: Distorted lattice that feels organic and shifting.
- **HUD**: Pulsing indicators that resemble living organisms.
- **VFX**: Shadow tendrils, screen distortion, and purple lightning.

### Treasury (Shop)
- **Display**: 3-column Grid on "Altar" style panels.
- **Currency**: Forsaken souls and blinking eyes in the HUD.

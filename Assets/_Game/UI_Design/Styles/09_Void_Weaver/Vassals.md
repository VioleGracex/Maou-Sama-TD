# Vassals (Gallery) - Void Weaver Style

## Visual Aesthetic
- **Background**: A non-Euclidean space where units are held in gravity-defying stasis within purple energy fields.
- **Grid Layout**: Shifting shadow-panels where portraits pulse with a faint, sickly green light.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Shadow ripple arrow. |
| `BtnUnitCard` | Unit Portrait | Unit Details | Bordered by pulsing purple fractal veins. |
| `BtnFilter` | Eye Icon | Filter Panel | A large, blinking cosmic eye icon. |
| `BtnBatchAction`| Claw Icon | Batch Selection| A shadow-claw icon. |

## Internal Windows & Overlays
- **Filter Panel**: A field of floating eyes that drift into position.
- **Unit Details Overlay**: Character stands in a void-spotlight. Stats are shown in pulsing purple glyphs.
- **Upgrade Window**: A sub-page where units are "merged" with shadow entities in a fractal circle.

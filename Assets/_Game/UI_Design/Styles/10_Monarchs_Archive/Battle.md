# Battle (Combat HUD) - The Monarch's Archive Style

## Visual Aesthetic
- **Environment HUD**: Polished mahogany frames with brass corner-caps.
- **Grid Tiles**: Blueprint-style lattice on a high-quality paper texture.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnPause` | II Icon | Pause Menu | Heavy brass clockwork gear. |
| `BtnSpeed` | >> Icon | N/A (Toggle) | Brass pendulum icons for speed levels. |
| `BtnAuto` | AUTO | N/A (Toggle) | A royal seal icon. |
| `BtnUnitIcon` | Unit Avatar | Deploy Unit | Portfolio-style portrait with brass trim. |
| `BtnSkill` | Skill Icon | Activate Skill | A wax seal that glows and "cracks" when ready. |

## Internal Windows & Overlays
- **Pause Menu**: A set of double mahogany doors that swing closed.
- **Victory Screen**: Elegant script "VICTORY DOCUMENTED" on a parchment banner.
- **Defeat Screen**: A closed book effect with soft, dark shadows.

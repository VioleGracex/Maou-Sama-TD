# Manifest (Summon Altar) - The Monarch's Archive Style

## Visual Aesthetic
- **Background**: A grand lectern in the center of the Archive, holding a massive, ancient tome.
- **Summon Altar**: The "Legendary Archive" book that releases golden light when opened.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Brass-bound parchment icon. |
| `BtnPull1` | Summon x1 | Summon Sequence| A single wax-sealed mandate. |
| `BtnPull10` | Summon x10 | Summon Sequence| A stack of ten leather-bound journals. |
| `BtnDetails` | Information | Details Window | Brass library catalog icon. |
| `BtnBannerNext` | Right Arrow | Next Banner | A page-flip transition sound/effect. |

## Internal Windows & Overlays
- **Summon Result Grid**: 2x5 Grid of leather-textured cards with brass corners.
- **Character Reveal**: A page of the central book flips rapidly, then character steps out of the final illustration.
- **Pity Tracker**: A brass bookmark that moves down a side-panel as pulls accumulate.

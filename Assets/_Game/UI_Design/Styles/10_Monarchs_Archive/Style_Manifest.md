# Style Manifest: The Monarch's Archive

## Core Visual Prompt
**Theme**: Regal, Historical, Grand Library.
**Colors**: Royal Burgundy (#800020), Deep Mahogany (#4D1919), Polished Brass (#B5A642).
**Aesthetic**: Leather-bound books, library shelves, brass fixtures, and wax seals.
**Typography**: "Libre Baskerville" - classic, traditional, feels like high-end print.

## Component Design
- **Buttons**: Brass-trimmed leather pads. Heavy mechanical "click" sound.
- **Panels**: Wood-paneled windows or opened book double-pages.
- **Icons**: Wax-sealed scrolls, stamped crests, and ink pens.

## Page Flows (Gacha Optimized)

### Citadel (Home)
- **Character Frame**: Behind character is a curved library wall.
- **UI Layout**: Formal, structured like a library catalog.
- **Interaction**: Page-flipping and heavy wood impact sounds.

### Conquest (Campaign Map)
- **Map Style**: Massive world globe or flat map on a library table.
- **Navigation**: Full-screen "Registry" overlay for stage selection.
- **Nodes**: Brass pins or library stamps.

### Manifest (Summon Altar)
- **Summon Sequence**: A massive "Legendary Archive" book opens with a burst of light.
- **Results Screen**: 2x5 Grid on wood-paneled or paper-textured panels.
- **Reveal**: Character steps out from a portrait in the book.

### Vassals (Gallery)
- **Layout**: Systematic "Catalog" Grid Scroll View for unit cards.
- **Filters**: Library stamp icons for sorting by category.
- **Stats**: Written in calligraphy on aged paper sidebars.

### Battle (Tactical Grid)
- **Grid**: Blueprint or tabletop map on a mahogany surface.
- **HUD**: Brass-rimmed boxes. Health bars are ribbons with wax seals.
- **VFX**: Paper flutters, ink splashes, and brass-light flashes.

### Treasury (Shop)
- **Display**: 3-column Grid representing the "Monarch's Registry".
- **Currency**: Royal seals and brass tokens on a leather-textured top bar.

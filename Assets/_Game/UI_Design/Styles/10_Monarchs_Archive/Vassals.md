# Vassals (Gallery) - The Monarch's Archive Style

## Visual Aesthetic
- **Background**: A high-end library catalog room with brass-capped shelves and soft golden light.
- **Grid Layout**: Units are displayed as portrait-cards on a mahogany-paneled grid with brass nameplates.

## Buttons & Navigation
| Button Name | Label / Icon | Redirects To | Visual Description |
| :--- | :--- | :--- | :--- |
| `BtnBack` | Back Arrow | Citadel (Home) | Brass quill pen icon. |
| `BtnUnitCard` | Unit Portrait | Unit Details | Mahogany-framed portrait with brass clips. |
| `BtnFilter` | Magnifier | Filter Panel | A brass-rimmed reading lens icon. |
| `BtnBatchAction`| Book Icon | Batch Selection| A leather-bound registry book icon. |

## Internal Windows & Overlays
- **Filter Panel**: A sliding drawer effect with labeled brass tabs.
- **Unit Details Overlay**: Character is shown as a full-page illustration in a large book. Stats are in calligraphy.
- **Upgrade Window**: A sub-page where units are "recorded" into history using ink and seals.

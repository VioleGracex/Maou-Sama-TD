# Maou-Sama TD: 10 Unique UI Style Prompts

These prompts are designed for AI image generation (e.g., Midjourney, Stable Diffusion) or as design briefs for artists to explore different "Unique and Simple" aesthetics for the game.

### 1. Astral Codex (Celestial Mystery)
> **Prompt**: "High-fidelity Gacha UI design. Theme: Astral Codex. The background is a moving celestial map with glowing constellations. UI elements are semi-transparent parchment with starlight-gold accents. Navigation links are represented as planetary orbits. Typography: Elegant, thin serif with glowing 'mana' particles. Aesthetic: Mystical, ancient, and airy."

### 2. Abyssal Forge (Primal Power)
> **Prompt**: "Gacha-style UI for a Demon Lord game. Theme: Abyssal Forge. Dark obsidian panels with glowing molten lava veins at the borders. Buttons have heavy beveled metallic edges with runic engravings. Pulsating heat distortion effects on active elements. Typography: Heavy, slab-serif industrial font. Aesthetic: Intimidating, powerful, and forged."

### 3. Ethereal Silk (Fluid Elegance)
> **Prompt**: "Modern mobile game UI design. Theme: Ethereal Silk. Layers of translucent floating ribbons on a deep violet gradient background. Minimalist glassmorphism panels with soft rounded corners. Highlights use fluid wave animations. Typography: Modern minimalist sans-serif with wide spacing. Aesthetic: Premium, smooth, and magical."

### 4. Blood Contract (Gothic Sovereign)
> **Prompt**: "Dark Gothic fantasy UI for 'Maou-Sama TD'. Theme: Blood Contract. Crimson ink splatters and parchment textures. Icons are styled as red sealing wax stamps on dark leather panels. Deep vignette and heavy shadows. Typography: Bold Blackletter combined with clean modern data fonts. Aesthetic: Sovereign, dangerous, and historical."

### 5. Runic Neon (Occult Cyber)
> **Prompt**: "Fusion UI design. Theme: Runic Neon. Cyberpunk UI hardware with glowing mystical runes. Matte black carbon-fiber panels contrasted with electric cyan and magenta neon energy lines. Glitchy transition effects. Typography: Sharp geometric mono fonts. Aesthetic: Tech-occult, futuristic, and high-energy."

### 6. Divine Marble (Ancient Authority)
> **Prompt**: "Gacha Home Screen UI. Theme: Divine Marble. Background of a grand marble palace hall. UI elements use ivory textures and intricate gold filigree framing. Layout is architectural and symmetrical. Typography: Classical Trajan-style serif for titles. Aesthetic: Grand, bright, and intimidatingly holy."

### 7. Monochrome Shade (Ink Wash Minimal)
> **Prompt**: "Stylized Gacha UI. Theme: Monochrome Shade. Entire interface is high-contrast black and white using ink-wash textures. Only character portraits are in full color to make them pop. Minimalist line-art icons and thin dividers. Typography: Elegant hand-drawn script. Aesthetic: Artistic, unique, and high-contrast."

### 8. Alchemical Laboratory (Steampunk Occult)
> **Prompt**: "Complex game UI. Theme: Alchemical Laboratory. Brass pipes, gauges with bubbling green liquid, and rotating clockwork gears. Interactive buttons are brass levers or glass dials. Sepia-toned paper backgrounds. Typography: Typewriter font and cursive script notes. Aesthetic: Scientific, curious, and steampunk."

### 9. Void Weaver (Quantum Horror)
> **Prompt**: "Abstract Gacha UI. Theme: Void Weaver. Deep space background with reality-tearing purple cosmic nebulae. Borders are flickering purple energy. Panels seem to warp or pulsate slightly. Icons are bioluminescent orbs. Typography: Glitchy, thin, glowing futuristic font. Aesthetic: Unearthly, vast, and mysterious."

### 10. The Monarch's Archive (Bibliographic Fantasy)
> **Prompt**: "Premium mobile UI. Theme: The Monarch's Archive. The entire UI is built as a heavy leather-bound tome. Transitions involve turning crisp parchment pages. Navigation tabs use silk bookmark ribbons. Typography: Mix of handwritten calligraphic scripts and bold serif. Aesthetic: Knowledgeable, royal, and immersive."

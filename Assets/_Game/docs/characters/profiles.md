# Legion Profiles: Master Art & Design Bible

This document is the final authority for all character visual assets. Use the **Global Style Guide** to maintain consistency across AI-generated art, then apply the **Specific Character Specs** for each game context.

---

## 0. Global Anime Style Guide (The Prompt Base)
Every AI prompt should start with or include these modifiers to ensure a unified "Maou-Sama" aesthetic:

**Core Style Prompt**: 
> "High-quality anime illustration, masterwork, clean cel-shaded, sharp outlines, vibrant colors, dynamic dramatic lighting, intricate details, Arknights art style, Zenless Zone Zero aesthetic, sharp focus, 8k resolution, cinematic composition, white background (for sprites) or blurred citadel background (for portraits)."

**Technical Constraints**:
*   **Perspective**: Characters must face slightly LEFT (towards the enemy path).
*   **Shadows**: Use high-contrast "2-tone" anime shading. No soft gradients on skin.
*   **Linework**: Consistent medium-thickness black outlines.

# UI Direction: The Sovereign's Authority (Fate/GO x Demon Lord)

## 1. Vision Statement
The new UI direction departs from the "Anime-Industrial" look of Arknights and moves towards a **Grand Gothic-Fantasy** aesthetic. It should feel like a powerful Demon Lord's ritual chamber—where authority is absolute, and fate is forged through ancient seals.

## 2. Key Pillars

### A. The "Authority Seals" (Core Motif)
- **Runic Geometry**: All major interactive elements (Summon buttons, Level Up, Skill Activation) are framed by circular or hexagonal runic arrays.
- **Mana Pulse**: Seals should have a subtle "breathing" glow. When clicked, the runes should rotate and flare.
- **Placement**: Used as backgrounds for character portraits, icons, and center-screen focal points.
- **Gameplay Link**: This motif represents the **Authority Seals** currency used during battle; their presence in the meta-UI serves to reinforce the Maou's omnipresent power.

### B. Fate/Grand Order Influence
- **Ornate Framing**: Use gold Filigree and thick, noble borders for character cards (Rarity-based: Silver, Gold, Platinum).
- **Typography Layout**:
  - Main Title: High-contrast Serif or Elegant Gothic font.
  - Sub-title: Smaller sans-serif with tracked-out spacing.
  - Demon Script: Add decorative "illegible" runic text for flavor.
- **Card Design**: Character portraits should feel like "Vassal Contracts"—rectangular cards with a visible rarity border and a signature "Class" icon (e.g., General, Mage, Assassin).

### C. Color Palette: "Abyssal Royalty"
- **Primary Background**: `#060609` (Obsidian)
- **Secondary Surface**: `#121218` (Midnight Blue-Black)
- **Primary Accent**: `#B8860B` (Dark Gold) - For borders and filigree.
- **Secondary Accent**: `#00D2D3` (Tech-Cyan) -> Transitioning to `#5D3FD3` (Royal Purple) or `#7DF9FF` (Electric Blue) for magical energy.
- **Danger/Action**: `#960018` (Carmine Red).

## 4. Hierarchy & Layout Rules
1. **Z-Space**: Use multiple layers of translucent glass with parallax runic overlays.
2. **Icons**: Minimalist but encased in ornate circular seals.
3. **Transitions**: Smoke and ink-splatter effects combined with golden "shimmer" particles.
4. **RectTransform**: All elements must use anchored RectTransforms for responsive scaling (as per Aris persona mandates).

# Strategic Roadmap

## Q1: Core Loop & Foundation (Current)
*   **Goal**: Playable prototype with 1 Chapter, 3 Characters, and basic Save/Load.
*   **Key Feature**: Character Creation & Basic Tower Defense.

## Q2: Content & Meta (Next)
*   **Goal**: 3 Chapters, 10 Characters, full Romance system.
*   **Key Feature**: 
    *   **Unit Upgrades**: Level up using "Mana Crystals".
    *   **Romance Events**: Visual Novel scenes triggered by usage count.
    *   **Daily Dungeon**: Rotating resource maps.

## Q3: Social & Polish
*   **Goal**: Public Beta release.
*   **Key Feature**:
    *   **Cloud Save**: Firebase integration.
    *   **Leaderboards**: Global score tracking.
    *   **Events**: "Summer Beach" event with limited banner.

## Q4: Monetization & Expansion
*   **Goal**: 1.0 Release.
*   **Key Feature**:
    *   **Shop**: IAP for Crystals (Stripe integration).
    *   **Guilds**: Basic chat and weekly raid boss.
